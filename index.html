Conmutacion de paquetes

1961
Leonard Kleinrock publicó desde el MIT el primer documento sobre la teoría de conmutación de paquetes

Arpanet

1969
se estableció la primera conexión de computadoras, conocida como ARPANET, entre tres universidades en California y una en Utah, Estados Unidos.

Modelo OSI

1989
Integración de los protocolos OSI en la arquitectura de Internet, facilitando el uso de distintos protocolos de comunicaciones.

W W W

1991
Tim Berners-Lee crea la World Wide Web (WWW) como un sistema para compartir información en línea.

Web 1.0

1991-2004
En esta fase, la web consistía principalmente en páginas estáticas de texto y gráficos. La interacción era limitada y la mayoría de los sitios web eran de tipo informativo.

Google

1997
Google es fundado como un motor de búsqueda para la WWW.

Web 2.0

2004-2010
En esta fase, la web se convirtió en una plataforma más interactiva y colaborativa. Los usuarios podían interactuar con el contenido y entre ellos a través de herramientas como blogs, redes sociales y servicios de video.

Web 3.0

2010
En esta fase, la web se está convirtiendo en una plataforma más inteligente e interconectada. Se están desarrollando tecnologías como el Internet de las cosas (IoT), la inteligencia artificial y el aprendizaje automático para mejorar la experiencia del usuario y personalizar el contenido. Además, la web está evolucionando hacia una web semántica, en la que los datos se organizan y se conectan de manera más efectiva.

Inteligencia Artificial

2021
OpenAI lanza GPT-3, un modelo de lenguaje de inteligencia artificial altamente avanzado.

Web 4.0

Actualmente
algunos expertos han propuesto que la "Web 4.0" podría ser una web más inteligente, centrada en la personalización y la adaptabilidad, que aprovecha la inteligencia artificial, el aprendizaje automático y la realidad virtual/realidad aumentada para proporcionar experiencias web más inmersivas e interactivas.
También se ha sugerido que la "Web 4.0" podría involucrar una mayor integración entre la web y el mundo físico, por ejemplo, mediante el uso de sensores y dispositivos de Internet de las cosas para conectar el mundo físico con la web de manera más completa y eficiente.
